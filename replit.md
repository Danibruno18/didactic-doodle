# GymStyle & Humor

A React landing page for a gym clothing store (Brazilian Portuguese). The page showcases a product catalog with categories: camisetas, tops, calças, leggings, and bermudas.

## Project Structure

- `src/LandingPage.jsx` — Main landing page component
- `src/main.jsx` — React entry point
- `src/index.css` — Tailwind CSS imports
- `index.html` — HTML entry point
- `vite.config.js` — Vite configuration (port 5000, allowedHosts: true)
- `tailwind.config.js` — Tailwind CSS configuration
- `postcss.config.js` — PostCSS configuration

## Tech Stack

- React 18
- Vite 5
- Tailwind CSS 3

## Development

```bash
npm run dev
```

Runs on `0.0.0.0:5000`.

## Deployment

Configured as a static site. Build output goes to `dist/`.

```bash
npm run build
```
